CSS-color

Posted on 2014-09-04 Edited on 2023-11-24 In CSS Views: Disqus:

 

在指定顏色時可以使用HEX value、RGB value、color name

一、HEX value

其格式為#ff0000，

十六進制表示法以#字號開頭後接十六進制六位數字

例如：

#FFFFFF為白色

#FF0000為紅色

#00FF00為綠色

#0000FF為藍色

#000000為黑色

#FFFF00為黃色

#00FFFF為青綠色

另有三位數字格式

例如：

#FFF為白色

#F00為紅色

#0F0為綠色

#00F為藍色

#000為黑色

#FF0為黃色

#0FF為青綠色

 

二、RGB value

其格式為rgb(255,0,0)

#p1 {background-color: rgb(255,0,0);} / * red */
#p2 {background-color: rgb(0,255,0);} / * green */
#p3 {background-color: rgb(0,0,255);} / * blue */

另有一種RGBA格式，其多了透明度的掌控，值為0~1

#p1 {background-color: rgba(255,0,0,.1);} / * red */

 

三、color name

例如：red

<html>
