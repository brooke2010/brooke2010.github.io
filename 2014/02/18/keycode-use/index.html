實作可以每輸入一字元並且更正一字元功能(KeyCode用法)

Posted on 2014-02-18 Edited on 2023-11-24 In VB.NET Views: Disqus:

 

於已輸入字串，實作可以每輸入一字元並且更正一字元功能，sample code如下

    Private Sub txtA_KeyDown(sender As Object, e As System.Windows.Forms.KeyEventArgs) Handles txtA.KeyDown
        eKey = e.KeyCode
    End Sub
 
    Private Sub txtA_TextChanged(sender As System.Object, e As System.EventArgs) Handles txtA.TextChanged
        If eKey = "46" Then '避掉無限迴圈刪除鍵
            MsgBox("DEL")
        Else
            System.Windows.Forms.SendKeys.SendWait("{DEL}") '刪除下一個字元
        End If
    End Sub
