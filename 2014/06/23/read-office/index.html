當程式要去讀取office檔而作業系統沒有灌 office2007/2010/2013時

Posted on 2014-06-23 Edited on 2023-11-24 In OS重灌 Views: Disqus:

 

通常已經安裝office2007/2010後，

已經不用而外安裝「Microsoft Access Database Engine 2010 可轉散發套件」，

就可以讓程式利用OLEDB或ODBC方式連線至OFFICE檔案讀取資料，

所以當程式要去讀取office檔而作業系統沒有灌 office2007/2010/2013時，

請安裝「Microsoft Access Database Engine 2010 可轉散發套件」即可由程式讀取OFFICE檔案。

 

參考資料：

[AccessDataSource] Office Access 2010 / 2013 與 Microsoft.ACE.OLEDB

檢查是否已經安裝 「Microsoft Access Database Engine 2010 可轉散發套件」驅動程式；Microsoft Access Database Engine 2010 Redistributable
