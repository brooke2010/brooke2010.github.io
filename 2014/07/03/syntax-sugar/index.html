一、?? null-coalescing operator
例如：int y = x ?? -1;
意指將x值指定給y，但如果x值為null時，則將-1指定給y。

二、??= null-coalescing assignment operator

意指如果 x 為 null 時，則將 “a” 指定給 x。
另一例為，如果 y 不為 null 時，則不需將 “c” 指定給 y。
