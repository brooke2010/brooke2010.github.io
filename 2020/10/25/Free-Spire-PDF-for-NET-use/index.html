Free Spire.PDF for .NET 的使用

Posted on 2020-10-25 Edited on 2023-11-24 In 工具教學 Views: Disqus:

 

Free Spire.PDF for .NET 是 Spire.PDF for .NET 的免費版本，無需購買即可用於個人或商業用途。

使用該組件，程序員可以 在 .NET( C#, VB.NET, ASP.NET, .NET Core) 程序中創建、讀取、寫入、編輯和操作 PDF 文檔。

Free Spire.PDF for .NET 支持的功能十分全面，例如文檔安全性設置（電子簽名），

提取PDF 文本、附件、圖片，PDF 合併和拆分，更新Metadata，設置Section，

繪製圖形、插入圖片、表格製作和加工、導入數據等等。除此以外，

Free Spire.PDF 還可以將 TXT 文本、圖片、HTML 高質量地轉換為 PDF 文件格式。

友情提示：免費版有 10 頁的頁數限制，在創建和加載 PDF 文檔時要求文檔不超過 10 頁。

將 PDF 文檔轉換為圖片時，僅支持轉換前 3 頁。我們僅對免費版進行不定期維護。

 

用法如下：

一、請先把以下兩檔案加入參考

 

二、範例程式

using System;
using Spire.Pdf;

namespace ConsoleApp1
{
    class Program
    {
        static void Main(string[] args)
        {
            PdfDocument doc = new PdfDocument();
            doc.LoadFromFile("C:\\123.pdf");
            //使用預設印表機列印文件所有頁面
            doc.Print();
        }
    }
}

 

備份檔下載：spire.pdf-fe_6.10
