node.js Hello World

Posted on 2020-06-14 Edited on 2023-11-24 In node.js Views: Disqus:

 

本文章會 step by step 方式來 Run 起一個 node.js Hello World

 

一、安裝 node.js

想藉由 node.js 把一個 server 架起來，前提之下，當然是先安裝好 node.js 囉

 

二、創立專案資料夾 myProject，並把 hello-world.js 增加到專案資料夾裡面

如下圖

 

三、hello-world.js 的內容為

const http = require('http');

const hostname = '127.0.0.1';
const port = 3000;

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hello, World!\n');
});

server.listen(port, hostname, () => {
  console.log(`Server running at http://${hostname}:${port}/`);
});

 

四、Run 起 server

於 myProject 資料夾裡執行如下指令

$ node .\hello-world.js
