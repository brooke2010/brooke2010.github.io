express.js 使用 ejs 模版引擎

Posted on 2020-09-06 Edited on 2023-11-24 In node.js Views: Disqus:

 

本文章將會 step by step 於 express.js 專案裡去套用 ejs 模版引擎

 

一、先準備好 express.js 空白專案

 

二、安裝模版引擎套件

$ npm install ejs

 

三、於 app.js 裡，將模版引擎改成 ejs

 

四、於 routes\index.js 裡的設定可知，傳入模版引擎的參數名稱為「title」

 

五、把 views 裡不相關的檔案拿掉，並只留一個 index.ejs 檔名

其內容為

<!DOCTYPE html>
<html>

<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width">
</head>

<body>
  <p><%= title %></p>

</body>

</html>

 

六、把 server run 起來

執行指令

$ node .\bin\www

執行結果為
