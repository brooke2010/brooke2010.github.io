LinqPad 的使用

Posted on 2021-04-26 Edited on 2023-11-24 In 工具教學 Views: Disqus:

 

設定連線



LINQPad有三種編寫方式

一、Expression: 單行程式或是單個LINQ陳述式

範例1、

DateTime.Now.ToString("yyyy/MM/dd")


範例2、

以北風資料庫為例

from x in Products

select x.ProductName


二、Statement(s): 多行的程式片段

var w = from x in Products

select x.ProductName;

w.Dump();


三、Program: 需要撰寫Class或是Method時

void Main()
{
    Test t = new Test();
    t.Hello().Dump();
    World().Dump();
}

string World(){
    return "World";
}

class Test{
    public string Hello(){
        return "Hello";
    }
}


四、Program: 取用資料表

void Main()
{
    Customers.Where(x => x.City == "Boston").Dump();
}
