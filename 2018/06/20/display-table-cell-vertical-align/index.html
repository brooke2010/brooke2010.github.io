使用 display:table-cell; 來做垂直置中

Posted on 2018-06-20 Edited on 2023-11-24 In CSS Views: Disqus:

 

不曉得我之前在糾結什麼，在做垂直置中時，會刻意閃避此方法實作，

現在我不得不承認，使用 display:table-cell; 來對一個區塊做垂直置中的確是一個好方法之一。

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width">
    <style>
        .main-header {
            display: table;
            height: 100vh;
            width: 100%;
        }
        .main-header-content {
            display: table-cell;
            vertical-align: middle;
            text-align: center;
        }
    </style>
</head>
<body>
    <article class="main-header">
        <header class="main-header-content">
            <h1 class="page-title">PRO SHIEH</h1>
            <h2 class="page-description">do me best, make you want.</h2>
        </header>
    </article>
</body>
</html>
