使用迴圈來執行 SQL

Posted on 2022-07-17 Edited on 2023-11-24 In T-SQL Views: Disqus:

 

想要使用迴圈來執行 SQL 時，可利用 while 語法，其結構如下

DECLARE @cnt INT = 0;

WHILE @cnt < cnt_total
BEGIN
  {...statements...}
  SET @cnt = @cnt + 1;
END;

 

範例：

DECLARE @cnt INT = 0;


WHILE @cnt < 3
BEGIN
   select ‘很重要，所以要說三次。’
   SET @cnt = @cnt + 1;
END;
